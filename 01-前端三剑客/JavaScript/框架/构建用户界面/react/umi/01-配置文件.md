# 1.define
* Type: ```object```
* Default: ```{}``
* 作用: 用于提供给代码中可用的变量。
* 例如：
```js
// .umirc.ts
export default {
  define: {
    FOO: 'bar'
  }
}

// 书写console.log(hello, Foo), 会被编译成console.log(hello, 'bar')
```
* 注意define对象的属性值会经过一次```JSON.stringify```的转换

* 内置的define属性
  * process.env.NODE_ENV, 值为development || production
  * process.env, 默认值为{NODE_ENV: development || production}
