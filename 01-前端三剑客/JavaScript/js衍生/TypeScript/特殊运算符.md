# 1. ??空值合并运算符
* 当左侧操作数为 null 或 undefined 时，其返回右侧的操作数，否则返回左侧的操作数;
* 与逻辑或 || 运算符不同，逻辑或会在左操作数为 false 值时返回右侧操作数。

# 2. &
* &表示交集，两个类型都拥有的类型
```ts
type Env1 = 'prod' | 'test' | 'dev';
type Env2 = 'prod' | 'boe' | 'ppe';
type EnvInter = Env1 & Env2; // 'prod'

```
