# 1.圆
```js
 var cnv = document.getElementById("canvas");
  var cxt = cnv.getContext("2d")
	// 1.开始一个新路径
  /**
   * 判断开始一个新路径的唯一标准是“是否使用beginPath()方法”
   * 使用以下方法只是绘制图形，并不会开始新路径：moveTo()、lineTo()、strokeRect()、fillRect()、rect()、arc()、arcTo()、quadricCurveTo()和bezierCurveTo()
   * Canvas中的绘制方法，如stroke()、fill()等，都是以“之前最近的beginPath()”后面所有定义的状态为基础进行绘制的
   */
	cxt.beginPath();

  // 2.画圆

  /**
   * cxt.arc(x,y，半径，开始角度，结束角度， anticlockwise);
   * x和y:x表示圆心横坐标，y表示圆心纵坐标。也就是说，(x , y)表示圆心坐标。
   * 开始角度、结束角度：开始角度和结束角度都是以弧度为单位。在实际开发中推荐这种写法：度数*Math.PI/180
   * anticlockwise是一个布尔值，当其值为true时，表示按逆时针方向绘制；当其值为false时，表示按顺时针方向绘制。默认为false
   */
  cxt.arc(80, 80, 50, 0, 360*Math.PI/180, true);
  // 3.关闭当前路径，closePath()方法的作用是连接起点与终点，使其成为一个封闭的图形，也就是“关闭路径”
  cxt.closePath();
  // 描边
  cxt.strokeStyle = "HotPink";
  cxt.stroke();
	// 填充
  cxt.fillStyle = "HotPink";
  cxt.fill();
```

# 2.弧线
```js
var cnv = document.getElementById("canvas");
var cxt = cnv.getContext("2d")
// 方式1：arc
cxt.beginPath();
cxt.arc(80, 80, 50, 0, 360*Math.PI/180, true);
cxt.strokeStyle = "HotPink";
cxt.stroke();
// 方式2：arcTo
/**
 * 1. arcTo()方法就是利用开始点、控制点和结束点所形成的夹角，绘制一段与夹角的两边相切并且半径为radius的圆弧。
 * 2. 弧线的起点是“开始点所在边与圆的切点”，
 * 3. 弧线的终点是“结束点所在边与圆的切点”。
 * 4. 使用arcTo()方法绘制的弧线是两个切点之间长度最短的那个圆弧。
 * 5. 此外，如果开始点不是弧线起点，arcTo()方法还将添加一条当前端点到弧线起点的直线线段。也就是说，开始点坐标并不一定是弧线的起点坐标！
*/
// 1.
cxt.moveTo(20,20);
cxt.lineTo(70,20);
/**
 * arcTo(cx,cy,x2,y2,radius);
 * (cx , cy)表示控制点的坐标，
 * (x2 , y2)表示结束点的坐标，
 * radius表示圆弧的半径。
 * 开始点一般由moveTo()或lineTo()提供
 */
cxt.arcTo(120,20,120,70,50);
cxt.lineTo(120,120);
cxt.stroke();

```

# 3. 二次贝塞尔曲线
```js
var cnv = document.getElementById("canvas");
var cxt = cnv.getContext("2d");
cxt.moveTo(30, 120);
/**
 * quadraticCurveTo(cx, cy, x2, y2);
 * 绘制一条二次贝塞尔曲线，我们同样也需要3个点的坐标：开始点、控制点和结束点
 * (cx , cy)表示控制点的坐标，
 * (x2 , y2)表示结束点的坐标。
 * 开始点一般由moveTo()或lineTo()提供
 */
// 画曲线
cxt.quadraticCurveTo(100, 20, 160, 120);
cxt.stroke();
```
# 4. 三次贝塞尔曲线
```js
var cnv = document.getElementById("canvas");
var cxt = cnv.getContext("2d");
cxt.moveTo(30, 120);
/**
 * bezierCurveTo(cx1, cy1,cx2,cy2, x3, y3);
 * 绘制一条三次贝塞尔曲线，我们同样也需要3个点的坐标：开始点、控制点1、控制2和结束点
 * (cx1 , cy1)表示控制点1的坐标，
 * (cx2 , cy2)表示控制点2的坐标，
 * (x3 , y3)表示结束点的坐标。
 * 开始点一般由moveTo()或lineTo()提供
 */
// 画曲线
cxt.bezierCurveTo(20, 20, 120, 120, 120, 60 );
cxt.stroke();
```
