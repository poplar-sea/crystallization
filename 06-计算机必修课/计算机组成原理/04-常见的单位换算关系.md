# 1. 字节的换算关系：
* 1字位(bit) = 1个二进制数
* 1字节(Byte) = 8位(bit) = 8个二进制数
* 1B = 8b
* 1KB = 1024B
* 1MB = 1024KB
* 1GB = 1024MB
* 1TB = 1024GB
* 1PB = 1024TB
* 1EB = 1024PB
* 1ZB = 1024EB
* 1YB = 1024ZB
* 1BB = 1024YB

# Hz(赫兹)的换算关系：
* 周期（秒）= 1 / 频率（Hz）
* 频率（Hz）= 1 / 周期（秒）
* 1Hz表示每秒振动1次
* 1赫兹（Hz）=1个周期
* 1千赫（kHz）=1000Hz
* 1兆赫（MHz）=1000kHz=1000000Hz
* 1吉赫（GHz）=1000MHz=1000000kHz=1000000000Hz
