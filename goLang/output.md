# golang语言的字符使用的是utf-8编码
* 英文是一个字节，中文是三个字节
# ftm.Printf 格式化输出
* %T 输出Go语言语法格式的类型和值
* %c 输出字符类型
* %p 输出变量取地址后的指针值
* %v 输出值得本来值
* %+v 在%v的基础上，对结构体字段名和值进行展开
* %#v 输出Go语言语法格式的值
* %b 整形以二进制方式显示
* %o 整形以八进制方式显示
* %d 整形以十进制方式显示
* %x 整形以十六进制方式显示
* %X 整形以十六进制方式、字母大写显示
* %U Unicode字符
* %f 浮点数
# 字符存储到计算机中，需要将字符对应的码字（整数）找出来
* 存储：字符 -> 对应码值 -> 二进制 -> 存储
* 读取：二进制 -> 码值 -> 字符 -> 读取
