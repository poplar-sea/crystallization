# 1.数组
## 1.1 `...`的用法
### 1.1.1 用于函数有多个不定参数的情况，可以接受多个不确定数量的参数
```go
func test1(args ...string) { //可以接受任意个string参数
    for _, v:= range args{
        fmt.Println(v)
    }
}
```
### 1.1.2 切片(slice)可以被打散进行传递
```go
func main(){
var strss= []string{
        "qwr",
        "234",
        "yui",
        "cvbc",
    }
    test1(strss...) //切片被打散传入
}
```
### 1.1.3 标识数组元素个数
```go
//这里，...意味着数组的元素个数：
stooges1 := [...]string{"Moe", "Larry", "Curly"} // len(stooges1) == 3
//上面的例子与下面的等效
stooges2 := [3]string{"Moe", "Larry", "Curly"} // len(stooges2) == 3
```
